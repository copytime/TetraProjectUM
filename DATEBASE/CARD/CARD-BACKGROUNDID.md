#描述
卡牌的背景图片 id。
此 id 决定了这张卡牌使用哪个背景来显示。
同时也影响了 buff 图标。
此 id 对应目录为：
`游戏根目录\TetraProject_Data\StreamingAssets\Packages\Builtin\CardBackground`
#示例
修改 `子弹` 卡牌的背景为橙色。
把 id 为 60 的子弹的 `backgroundId` 表头列修改为 `4`。
