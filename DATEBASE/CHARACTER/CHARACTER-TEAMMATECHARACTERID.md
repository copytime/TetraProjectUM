；2019/5/12 上午 2:12:37
#描述
角色的队友 id。
为标记是 `Main` 的角色创建队友。
region w { 注意
没有 `Main` 标签的角色不能使用队友。
即不是主角(玩家主要操作角色)不能使用队友。
} region
#可用选项
+ 所有 Character 子列表中的角色 id
#示例
把 `红莲` 的 `teammateCharacterId` 改为 4。
