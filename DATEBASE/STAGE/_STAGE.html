　└◆ Stage 子列表

◇ 描述

此子列表用于存放游戏关卡。

◇ 简要

表头简要介绍
id 关卡 id，全游戏唯一
displayName 卡卡的显示名称
//comment 注释类信息不会被游戏读取：关卡的注释
section 关卡的区域，大关卡
part 关卡的部分，小关卡
mapId 关卡的地图 id
initCode 关卡初始化代码
mobId[0~5] 关卡的怪物 id
rewardMoney 关卡的奖励
musicId 关卡的背景音乐 id
